Bienvenu sur XML-VALIDATOR

Comment ça marche ?

Vous pouvez valider vos fichers xml de plusieurs façons.

1-Dtd validation :
1.1 -La validation de dtd interne .

Dans ce cas, vous devez écire le contenu de votre document dans le champs qui vous ait fourni pour le valider.

1.2 -La validation d'un document xml avec un dtd externe.

Dans ce cas, vous devez selectionner le fichier xml et dtd externe pour valider.

2-XmlShémas validation :
2.1 -La validation d'un document xml avec un schéma.

Dans ce cas, vous devez selectionner le fichier xml et le schéma pour valider.
